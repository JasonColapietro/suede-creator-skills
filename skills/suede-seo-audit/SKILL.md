# Suede SEO Audit

Use this skill to audit a Suede public surface for discoverability, trust, and
conversion without turning the page into keyword stuffing. In this skill, SEO
includes AEO and AI EO: search snippets, answer engines, AI summaries,
sourceable claims, schema, internal links, and citation-ready proof.

## Workflow

1. Verify the target URL or source file before recommending changes.
2. Identify the primary reader, search intent, answer intent, and one action
   the page should earn.
3. Check title, meta description, canonical URL, robots/indexability, sitemap,
   H1, heading hierarchy, Open Graph, Twitter card, image alt text, and JSON-LD.
4. Check whether durable Suede terms appear naturally: creator ownership,
   programmable IP, rights, provenance, registry-backed media, royalty routing,
   licensing readiness, agent commerce, Codex skills, and SKILL.md.
5. Check internal links, install links, proof links, answer-ready summaries,
   and dead or misleading links.
6. Replace vague public claims with concrete artifacts, commands, docs, scripts,
   URLs, examples, or verification steps.

Do not mark an audit complete unless the current page or source was inspected.
