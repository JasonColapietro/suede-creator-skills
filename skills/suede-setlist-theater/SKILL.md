# Suede Setlist Theater

Use this skill to turn a setlist into a show with shape.

## Stage

1. Identify the songs, venue type, set length, audience familiarity, and release
   goal.
2. Build the set arc:
   - entrance;
   - first hit;
   - breath;
   - reveal;
   - fan participation;
   - peak;
   - reset;
   - exit or encore.
3. Add transitions, spoken lines, visual cues, lighting notes, and crowd
   prompts.
4. Connect merch, fan rituals, QR actions, and post-show CTAs without making the
   show feel like an ad.
5. Keep safety, venue, and rights constraints visible.

## Output

```text
Set arc:
Song order:
Transitions:
Crowd moments:
Visual cues:

Post-show CTA:
```
