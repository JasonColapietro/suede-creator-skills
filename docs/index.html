What it does

01 Portable memory

Keeps preferences in linked skill context instead of bloating one giant prompt until the agent forgets the job

02 Suedify

Give it a reference URL and a target URL. It pulls the aesthetic grammar into your own site with safe assets and QA

03 Agent teams

Runs scout, build, review, verifier, and release lanes that share context instead of acting like siloed subagents

04 Copy + GEO

Distills copy, CTAs, metadata, schema, FAQs, and answer-ready proof for Google, Gemini, and model-readable answer surfaces

05 Code review

Runs deep review and A-F grading across correctness, security, data, UX, release behavior, and public-claim truth

06 Cue Suede

Tell it what you like or hate mid-run or at the end. The next pass uses the correction instead of vague memory

07 GitHub Pages proof

The public site is part of the product: polished docs, skill pages, copy bank, install paths, schema, sitemap, OG, and attribution

08 Music + IP suite

Builds campaign systems, release lint reports, rights passports, metadata checks, sync prep, provenance, credits, split notes, and intake reports

09 iOS set builder

Turns a site or product idea into iOS-ready copy, screenshots, App Store metadata, native-contract checks, and release QA

10 Visibility score

Scores brand and page visibility across first-screen clarity, CTA pull, proof, AI readability, schema depth, and search result fit

11 Project grade

Ends with blunt grades, missing gates, evidence, and the next fix so the agent does not hand back a pretty but unverified mess

12 MCP tools

Adds structured discovery, install options, SEO/AEO/GEO/AI EO audit scaffolds, QA checklists, and tool validation when structure helps
