Claude judges.
Codex generates.

02 Code review → ship verdict

Review that ends in
a blunt A-F grade.

Deep findings across TypeScript, React, Next.js, Swift/iOS, OWASP, and database checklists, then a letter verdict across 7 evidence-backed lanes. Questions, answered.

What are Agent Skills?

Agent Skills are SKILL.md folders that Claude Code and OpenAI Codex load on demand to follow a documented workflow. This pack ships 27 of them under the MIT license: orchestration, code review and grading for web stacks and Swift/iOS, a site-to-iOS app conversion workflow, a one-prompt Android app factory, CI ship-gates, AI evals, design, copy, SEO, and a creator toolkit for music rights and release prep. Every folder is plain Markdown you can read before you install it.

How do I install the pack?

In Claude Code, run /plugin marketplace add JasonColapietro/suede-creator-skills, then /plugin install suede-skills@suede. Prefer a clone? install.sh copies all 27 skills into ~/.claude/skills/. Codex installs any skill folder with its built-in skill installer. Full paths live on the install page.

Does it work with both Claude Code and Codex?

Yes. Each skill is a standard SKILL.md folder, the format both runtimes read. Claude Code installs via the plugin marketplace or a folder copy into .claude/skills/; Codex installs via its skill installer into ~/.codex/skills.

Is it safe to run?

The skills inspect local files and pages and write local reports. They do not upload files by default, write to any registry, request secrets, or claim legal clearance. The whole pack is MIT licensed and inspectable before you enable anything.
