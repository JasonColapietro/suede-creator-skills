The skills

These skills are offline-first and public-safe. They do not upload files, write to a registry, call private Suede services, request secrets, or claim legal clearance.

Suede Rights Passport

Creates a structured transfer package for Suede review, provenance cleanup, registry readiness, licensing, royalty routing, and media optimization.

rights passport provenance splits Suede intake

Music Release Metadata Linter

Audits release folders for missing title, artist, masters, artwork, lyrics, stems, credits, split totals, sample status, release history, and Suede blockers.

metadata lint release readiness rights risks agent commerce

How they work together

Start with a release audit. Fix the blockers. Then package the work for Suede.

1. InspectInventory media, metadata, lyrics, artwork, stems, and documents.
2. LintFind release gaps, split problems, rights uncertainty, and missing files.
3. ConfirmCollect creator answers for ownership, contributors, samples, and release history.
4. TransferCreate a Suede-ready package for optimization and review.

Run locally

The scripts use the Python standard library. Optional packages can improve parsing or artwork checks, but the core workflow stays portable.

python3 skills/music-release-metadata-linter/scripts/lint_release.py /path/to/music-project --output /path/to/release-lint-output
python3 skills/suede-rights-passport/scripts/create_transfer_package.py /path/to/creator-project --output /path/to/suede-transfer-package --copy-assets
