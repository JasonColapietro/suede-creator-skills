Search-ready answers

Suede workflow FAQ.

Short answers for builders, designers, agencies, creators, developers, and agents deciding whether the public Suede skill pack fits their workflow.

What are Suede Creator Skills?

Suede Creator Skills are free Codex and Claude skill folders for Suedify website restyling, design QA, copywriting, visibility grading, code grading, SEO/AEO/AI EO audits, agent-team QA, artist campaigns, and creator workflows.

Which website workflows do the skills support?

The skills support reference-site mimicry, website redesign, copywriting, visibility and CTA grading, SEO/AEO/AI EO metadata, public docs, launch packaging, A-F code grading, QA, artist campaign packaging, and creator utilities.

When should I use Agent Teams?

Use Agent Teams when work is large, risky, public, cross-surface, release-bound, design-heavy, or likely to collide with user or other-agent WIP. It scouts first, splits safe lanes, reviews the result, verifies release truth, and records evidence.

Do the skills copy websites exactly?

No. Suedify studies layout, rhythm, typography, color roles, motion, and proof structure, then recreates the design grammar with target-owned content and assets.
