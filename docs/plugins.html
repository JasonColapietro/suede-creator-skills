Plugins and MCP

Install Suede skills as plugins. Use MCP when it helps.

Suede skills can load as local Codex plugins, plain skill folders, or a local MCP server. The MCP is not decoration: it gives agents structured access to the skill catalog, install options, SEO copy audit scaffolds, and multi-lane QA checklists.

Suede Workflow Skills

Design, copy, review, agent teams, site alchemy, and two-URL restyling for production Suede work. This command assumes the local personal marketplace has the plugin source registered.

codex plugin add suede-workflow-skills@personal

MCP server: suede_workflow_mcp

Suede Creator Skills

Music release metadata linting and Suede Rights Passport transfer packages for creator folders. Public users can also clone the repo and copy the skill folders directly.

codex plugin add suede-creator-skills@personal

MCP server: suede_creator_mcp

Public source fallback

The @personal commands are local-personal marketplace installs. When that marketplace is not configured, use the public source path instead.

git clone https://github.com/jasoncolapietro/suede-creator-skills.git
cd suede-creator-skills
node mcp/suede-skills-mcp.mjs --profile creator

What the MCP adds

Skill discoveryAgents can ask for the right Suede skill instead of guessing from memory.
Install optionsCodex plugin, MCP, and skill-copy paths are returned as structured guidance.
SEO copy auditsThe MCP provides a full Suede SEO audit scaffold for pages, repos, README files, and skill docs.
QA checklistsAgents can generate a full QA pass for plugin, MCP, skill, docs, and public site changes.

MCP tools

list_suede_skillsLists Suede skills by workflow, creator, or all areas.
get_suede_skillReturns one skill description and when to use it.
suede_install_optionsReturns plugin, MCP, Codex, and skill-copy install options.
suede_copy_seo_auditBuilds an SEO copy audit scaffold with metadata, schema, CTA, proof, and verification checks.
suede_qa_checklistBuilds a multi-lane QA checklist for MCP, plugin, skill, docs, and public site work.

Run locally

The public repo includes the same dependency-free stdio MCP server used by the plugins.

node mcp/suede-skills-mcp.mjs --profile all

Profiles: all, workflow, or creator.

Safety boundary

The MCP returns catalog, install, audit, and QA context. It does not upload music, clear rights, write to a registry, distribute releases, approve payouts, or make public claims on its own.
