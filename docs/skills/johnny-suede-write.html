Public install command

Claude Code — project-level copy:

mkdir -p .claude/skills
cp -R /tmp/suede-creator-skills/skills/johnny-suede-write .claude/skills/

Or install the full pack: /plugin marketplace add JasonColapietro/suede-creator-skills then /plugin install suede-skills@suede.

Use your own company

Give the skill your company, audience, voice, proof, allowed claims, forbidden claims, CTA, assets, and reference URLs; it should keep the full quality workflow while replacing Suede voice with your brand.

Use $johnny-suede-write for my company instead of Suede voice.
Company: Acme Robotics.
Audience: operations leaders at mid-market manufacturers.
Voice: precise, practical, slightly bold, never hype.
Proof: customer quotes, uptime reports, product screenshots, install docs.
Forbidden claims: no guaranteed savings, no unverified rankings, no fake logos.
Primary CTA: book a workflow review.

What it covers

  • Public copy, docs copy, GitHub README copy, CTAs, FAQs, launch copy, product and mobile conversion copy, and product listing copy.
  • Founder voice, company voice, public explainer language, social posts, emails, and line edits.
  • Anti-slop scoring across directness, rhythm, trust, specificity, authenticity, density, and Search/AI readability.
  • SEO/AEO/AI EO titles, descriptions, headings, answer-ready summaries, FAQ structure, and discoverability language.
  • Claim boundaries for unsupported payouts, registry writes, legal clearance, rankings, partners, or private access.

Best prompt

Use $johnny-suede-write to rewrite this page for my company, tighten the CTAs, audit SEO/AEO/AI EO, handle product and mobile copy if relevant, run the full anti-slop line edit, and package final copy with claim boundaries.
