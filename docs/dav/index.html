What it does

Copy + SEO

Rewrites headlines, CTAs, metadata, FAQs, and answer-ready summaries with an anti-slop score

Visibility grader

Grades findability, first-screen clarity, CTA pull, proof, AI readability, and design signal from A to F

Code grader

Reviews correctness, security, state, tests, deploy readiness, and public-claim truth before ship

Agent teams

AI evals

Creator campaigns

Rights utilities
